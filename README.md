# qUeueNO

An interactive terminal-based educational game that demonstrates queue data structure operations through engaging gameplay.

## About

This project was created as part of my Data Structures course. It helps students understand queue operations (enqueue, dequeue, peek) through a fun, interactive game.

## Built With

- Java
- Data Structures & Algorithms

## How to Run

1. Clone the repository
2. Compile: `javac Main.java`
3. Run: `java Main`
