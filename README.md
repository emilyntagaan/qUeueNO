## About the Project

**qUeueNO** is an interactive terminal-based game that demonstrates queue data structure operations through engaging gameplay. The project emphasizes clean program structure, efficient data handling, and creating an engaging user experience within a text-based interface.

### Project Goals

- Demonstrate practical implementation of queue data structures
- Showcase logical flow and strategic decision-making
- Create an engaging terminal-based gaming experience
- Apply algorithmic thinking to game mechanics

## Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- Terminal/Command Prompt
- IntelliJ IDEA (recommended) or any Java IDE

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR_USERNAME/qUeueNO.git
   ```

2. **Navigate to project directory**
   ```bash
   cd qUeueNO
   ```

3. **Compile the program**
   ```bash
   javac Main.java
   ```

4. **Run the game**
   ```bash
   java Main
   ```

## How to Play

### Game Objective

Navigate through challenges using queue-based mechanics where timing and order matter!

### Controls

```
[Enter] - Confirm action
[1-9]   - Select options
[Q]     - Quit game
```

### Gameplay Tips

- Plan your moves ahead of time
- Remember: Queue follows FIFO (First In, First Out)
- Timing is crucial!
- Watch your score and try to beat your record

---

### Algorithm Design

- **Input Validation**: Ensures all player inputs are valid
- **State Management**: Tracks game progress and player score
- **Turn Logic**: Manages game flow and player actions
- **Win Conditions**: Determines success based on queue state
